## Как заливать свою хуйню

1. Создаете свой форк (кнопка есть в гитхабе)
2. Клонируете свой форк(качаете)
3. Делаете свое говно
4. Пушите отдельную ветку
5. Идете на гитхаб и видите кнопочку, которая позволит сделать ПР. Нажимаете
6. Описываете что вы сделали
7. Ждете одобрения
