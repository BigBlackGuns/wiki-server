## Как заливать свой код

1. Создаете свой форк (кнопка есть в гитхабе)
<img width="1170" height="178" alt="image" src="https://github.com/user-attachments/assets/06a95bf1-3540-4c0d-99c7-88a98b63da49" />

---

2. Клонируете свой форк(качаете)

Используйте команду, чтобы склонировать(скачать) свой форк.
```
git clone ССЫЛКА
```

3. Перед началом кодинга

В интерфейсе гитхаба переходите на свой форк, ищите кнопку sync fork. Если она доступна - жмете. Это обновит ваш форк до последней версии оригинального репозитория.
Иначе вы будете работать со старым кодом, что может выйти в проблемы

<img width="891" height="259" alt="image" src="https://github.com/user-attachments/assets/a7edc17e-3cf9-4100-a01e-c81a6989c494" />

---

Затем переходим в консоль и пишем пару команд

> Скачиваем изменения с гита, если они были
```
git pull
```

> Создаем ветку для этих изменений. Это нужно, чтобы быть как крутые программисты и структуризировать свои изменения
```
git checkout -b НАЗВАНИЕ_ВЕТКИ_КОТОРОЕ_КРАТКО_ПОКАЗЫВАЕТ_ИЗМЕНЕНИЯ
```


4. Делаете свое говно
- Открываете скачанный репозиторий.
- Пишите код, который хотите сделать

5. Заливаете свой код на гит

> Показываем гиту свои изменения
```
git add .
```

> Записываем ваши изменения в текущую ветку(обязательно создайте ее перед этим!)
```
git commit -m "КРАТКОЕ ОПИСАНИЕ ВАШЕГО ИЗМЕНЕНИЯ"
```

> Заливаем изменения на гит
```
git push
```


6. Идете на гитхаб и видите кнопочку, которая позволит сделать ПР. Нажимаете
7. Описываете что вы сделали
8. Ждете одобрения
